### 一键开启虚拟副屏——开启副屏.bat
```bash
@echo off
rem 切换到目标目录
cd /d C:\Program Files\usbmmidd_v2

rem 执行命令
deviceinstaller64 enableidd 1

rem 显示完成信息
echo 操作完成！
```

### Windows系统通过SMB访问映射Linux共享盘重启电脑后使用——启动SMB后5秒重启.bat
```bash
@echo off
rem 开启
dism /online /enable-feature /featurename:SMB1Protocol-Client /All /norestart
dism /online /enable-feature /featurename:SMB1Protocol-Deprecation /All /norestart
timeout /t 5 /nobreak
shutdown /r /t 0
echo 系统重启中
```

### Windows系统通过SMB访问映射Linux共享盘重启电脑后使用——关闭SMB后5秒重启.bat
```bash
@echo off
rem 关闭
dism /online /disable-feature /featurename:SMB1Protocol-Server /norestart
dism /online /disable-feature /featurename:SMB1Protocol-Client /norestart
dism /online /disable-feature /featurename:SMB1Protocol-Deprecation /norestart
timeout /t 5 /nobreak
shutdown /r /t 0
echo 系统重启中
```

### 一键启动指定分辨率的远程桌面文件——办公台式机.rdp
用文本编辑器打开 `.rdp` 文件，添加以下参数：

```bash
screen mode id:i:1
desktopwidth:i:1920
desktopheight:i:1080
session bpp:i:24
```

| 参数                       | 格式与值         | 作用                               | 典型选项                                                     |
| -------------------------- | ---------------- | ---------------------------------- | ------------------------------------------------------------ |
| **`screen mode id:i:1`**   | `i:1` 或 `i:2`   | 控制远程窗口的显示模式             | `1`=全屏<br>`2`=窗口化                                       |
| **`desktopwidth:i:1920`**  | `i:[数值]`       | 设置远程桌面水平像素（分辨率宽度） | `1920` (1080p)<br>`2560` (2K)<br>`3840` (4K)                 |
| **`desktopheight:i:1080`** | `i:[数值]`       | 设置远程桌面垂直像素（分辨率高度） | `1080` (1080p)<br>`1440` (2K)<br>`2160` (4K)                 |
| **`session bpp:i:32`**     | `i:16`/`24`/`32` | 定义颜色深度（位/像素）            | `16`=增强色（65536色）<br>`24`=真彩色（千万色）<br>`32`=带透明通道的真彩色 |
